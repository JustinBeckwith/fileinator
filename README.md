# fileinator

![inator](http://i.imgur.com/Hc9u9wz.jpg)

Behold my latest inator! Generate files full of random bytes.  Good for tests. 

### Installation

`npm install -g fileinator`

### Usage

`fileinator make 10gb ./bigfile`
